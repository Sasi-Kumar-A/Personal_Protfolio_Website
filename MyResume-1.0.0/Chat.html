Live Group Chat

Real-Time Group Chat Application (Python + Flask)

The Real-Time Group Chat Application is a web-based communication platform developed using Python and Flask. It allows multiple users to join a shared chat room and exchange messages instantly through a clean and interactive user interface.

The application features a modern UI with smooth transitions, message bubbles, timestamps, and sender identification. Flask is used as the backend framework to manage routing, server logic, and message handling, while HTML, CSS, and JavaScript are used for the frontend interface.

Users can enter their name, join the group chat, and start communicating in real time. The system handles multiple users simultaneously and updates messages dynamically without requiring page reloads, providing a seamless chat experience.

This project demonstrates my understanding of Python backend development, Flask framework fundamentals, real-time communication concepts, and responsive UI design.

Key Highlights

Python • Flask • HTML • CSS • JavaScript

The project showcases practical implementation of server-client interaction, session handling, and real-time message updates. It reflects my ability to build full-stack web applications and design user-friendly interfaces suitable for modern web platforms.
